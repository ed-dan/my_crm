
## Run Project

### Installation 

- **git clone [https://Dadova@bitbucket.org/diploma2024/my_crm.git](https://Dadova@bitbucket.org/diploma2024/my_crm.git)**
- **npm install**
- **composer install**
- **copy .env.example .env**
- **php artisan key:generate**
- **php artisan migrate**
- **php artisan db:seed --class=DemoDataSeeder**
- **php artisan serve**
- **npm run dev**
